# kvstrings
kvstrings is a tool same as "strings" command in Linux

### Built With

* [![C][C]][C]

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/d4rk3agle/kvstrings
   ```
2. Make install script executable 
   ```sh
   chmod +x ./install.sh
   ```
3. Install
   ```sh
   sudo ./install.sh
   ```
